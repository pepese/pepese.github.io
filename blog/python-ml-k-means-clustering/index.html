k-means の実装

model の各項目は以下の通り。

  • cluster_centers_
    • クラスタの中心座標
  • labels_
    • 各点に付与されたラベル、つまり、クラスタリングの結果
  • inertia_
    • 各点からそれぞれが属するクラスタの中心までの距離の挿話
